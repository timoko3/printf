# Учебная работа "Printf" 

## Описание 
Целью данной работы являлось написание собственной версии функции "printf" стандартной библиотеки языка Си (**libs**). Реализация данной функции осуществлялась на ассемблере для архитектуры процессора x86-64. Моя собственная функции в финальном варианте, как и библиотечная функция должна получать на вход строку содержащую текст и спецификаторы формата. Все кроме спецификаторов должно быть выведено на экран без изменений, а спецификаторы заменены на аргументы передающиеся в функцию по правилам определенным для конкретного спецификатора. Мною была реализовано поддержка основных спецификаторов таких, как %s, %c, %o, %x, %d, %f. Символ процента можно напечатав его подряд дважды. Также для того, чтобы моя версия по сравнению с библиотечной имела расширенный функционал была реализована поддержка спецификатора %b, требующего для себя в качестве аргумента число и выводящего это число в двоичной системе счисления.
