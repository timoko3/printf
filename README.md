# Учебная работа "Printf" 

## Описание 
В рамках данного проекта реализована собственная версия функции **printf** из стандартной библиотеки языка C (**libc**). Реализация выполнена на ассемблере для архитектуры x86-64 в операционной системе linux ubuntu.

Функция принимает форматную строку, содержащую обычный текст и спецификаторы формата. Обычный текст выводится без изменений, а спецификаторы заменяются соответствующими аргументами, переданными в функцию, в соответствии с правилами форматирования.

Поддерживаются следующие спецификаторы:
- **%s** - строка
- **%c** - символ 
- **%d** - целое число 
- **%f** - число с плавающей точкой 
- **%o** - восьмеричное представление 
- **%x** - шестнадцатеричное представление

Дополнительно реализован расширенный спецификатор:
- **%b** -двоичное представление

## Реализация
### Деление на токены
На первом этапе функция разбирает входную строку и выделяет в ней токены двух типов:
- обычный текст
- спецификаторы формата

Далее каждый токен анализируется:
- если токен является обычным текстом, он копируется в выходной буфер без изменений
- если токен является спецификатором, он передаётся в обработчик

Основная сложность реализации связана именно с обработкой спецификаторов формата.

Для их обработки была создана **jump table** — массив из 256 элементов (по количеству возможных ASCII-символов), содержащий указатели на функции-обработчики. 

После обнаружения символа % следующий символ используется как индекс в таблице. По этому индексу извлекается адрес соответствующей функции, которая выполняет обработку спецификатора.

Если спецификатор не поддерживается, управление передаётся функции обработки ошибок.
